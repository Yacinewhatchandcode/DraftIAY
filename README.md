# Draft AIY - Sovereign Cognitive Workspace
An infinite canvas built to enforce a zero-illusion execution protocol, allowing users to map out thoughts while visualizing the underlying AI agent swarm logic in real-time.

## Features
- **True Glassmorphism Interface**: Deep void dark-mode styling with glowing bezier spline connections and neon telemetry.
- **Node-Based Idea Drafting**: Draggable, groupable StickyNotes via `@xyflow/react`.
- **AI Rephrasing Pipeline**: Select a note, hit Rephrase, and watch the mock Vagabond/Cognitive-Core strip out noise into pure keywords.
- **Swarm Brain Logger**: A transparent terminal tracking internal logic, executed actions, and trace logs to enforce the Zero-Illusion policy.

## Tech Stack
React, Vite, Tailwind/Native CSS, Framer-Motion (Animations), Lucide-React (Icons), ReactFlow (Canvas Manipulation).

## Running Locally
```bash
npm install
npm run dev -- --port 8000
```
